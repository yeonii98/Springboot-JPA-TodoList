# Springboot로 만든 TodoList 프로젝트

---
### 프로젝트 개요
* 일정을 추가하고 삭제할 수 있는 간단한 todolist (MVC구조)
---
### 기술
* IDE : Spring Tool Suite 4
* DBMS : MySQL 5.7
* Frontend : BootStrap 4, HTML, CSS
* Backend : Java, JavaScript, Spring Boot
---
### 기능
* 일정 추가
* 일정 삭제
* 일정 목록
